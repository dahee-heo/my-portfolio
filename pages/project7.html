Stamp

직원 출결관리 서비스입니다. 직원용 페이지와 관리자용 페이지로 나누어져 있으며 직원용 페이지에서는 출결 체크 및 기록을 확인할 수 있고 관리자 페이지에서는 출결 기록 확인 및 직원, 부서, 공지 등을 관리할 수 있습니다. React 라이브러리를 사용하여 클라이언트 개발을 하였고, express 프레임워크를 사용하여 서버페이지를 개발하였습니다.

Skill
React, Recoil, Stitches, MongoDB, Express
Role
Front-end, back-end, UI/UX design

직원 페이지
전체 페이지 반응형 UI 구현
  • 가볍고 빠른 성능을 위해 stitches 라이브러리를 활용하여 반응형 UI 구현
로그인, 회원가입 기능 개발
  • react-hook-form 라이브러리를 이용하여 validation 처리

  • Recoil을 활용하여 user 상태를 저장, 관리하고 token 값을 통해 로그인 검증 및 로그아웃 기능 구현
출근 기록 및 데이터 조회 기능 구현
어드민 페이지
출결 대시보드 데이터 조회

  • axios 라이브러리를 통해 백엔드와 통신하여 데이터 등록, 조회, 수정, 삭제 기능 구현

samesite 이슈
  • 쿠키가 제대로 저장되지 않아 로그인 후 새로고침하면 계속 로그인이 풀리는 이슈가 있었습니다.
  • 해당 프로젝트에서는 프론트와 백의 도메인이 달라 쿠키 저장 시 samesite를 none으로 줘야 했습니다. 그러려면 secure옵션을 true로 줘야했는데 이 옵션은 ssr인증(https)을 받아야 사용이 가능하여 현재 해당 옵션을 적용하지 못했습니다.
  • 이러한 이슈를 통해 쿠키 옵션 및 크로스 사이트에 대해 이해하게 되었습니다.
CSS-in-JS
