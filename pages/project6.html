LG MyCarAlarm service

LG CNS의 상용 프로젝트로, TV와 차량의 블랙박스 기기들을 연동하여 알람 및 메세지를 전송하고 확인할 수 있는 서비스입니다. React 라이브러리를 사용하여 클라이언트 개발을 담당하였습니다. 이 서비스는 세가지 도메인으로 나뉩니다. - TV와 블랙박스를 연동하는 프로바이더 웹 - 소셜 계정을 연동하고 TV에 메세지를 전송할 수 있는 모바일 웹 - TV, 블랙박스, 메시지 정보들을 관리할 수 있는 어드민 페이지

Skill
React, Typescript, SCSS
Role
Front-end

프론트엔드 개발

모바일 웹
휴대폰 인증 페이지 개발
  • 회원가입 및 회원정보 수정 시 휴대폰 인증하는 페이지 구현
  • api로 인증코드 전송 요청 후 타이머 기능 setInterval을 활용하여 custom hook으로 구현
알림 설정 페이지 개발
  • checkbox의 checked 값에 따라 데이터 변경 요청하여 기능 구현
회원 정보 수정 페이지 개발
  • 닉네임, 휴대폰 번호 변경 페이지 구현.
  • 닉네임 변경 시 휴대폰 번호 변경 불가, 인증 코드 시간 초과 시 수정 불가 등 다양한 조건의 validation 적용
프로바이더 웹
블랙박스 브랜드 별(아이나비, 파인뷰 등) 상단 문구 노출
  • 블랙박스 브랜드 별로 상단의 안내 문구 다르게 노출되도록 기능 추가
어드민 페이지
블랙박스 브랜드 별 문구 등록
  • 문구 등록할 수 있는 입력창 추가 개발

유지보수

모바일 웹
소셜 로그인 후 추가 정보(휴대폰 번호, 닉네임) 기입
  • 소셜 로그인 후 이전에는 이메일만 보이던 페이지에 휴대폰 번호와 닉네임 추가
  • 닉네임 중복확인 기능 추가

얻게된 점

SQL DB 구조에 대한 이해
  • 백엔드 개발자와 협업하면서 DB 데이터도 조회할 수 있어 외래키 등 관계형 데이터가 어떤식으로 구성되는지를 파악할 수 있었습니다.
