사진?

Mnworks web site

Date
2023. 06 ~ 2023. 08
Skill
Svelte-kit, SCSS, Mysql
Role
Front-end, back-end

에이전시에서 진행한 업무로 스틸시리즈(SteelSeries) 국내 총판 업체 mnworks의 공식 사이트 및 관리자 페이지를 개발하였습니다. Sveltekit 프레임워크를 사용하여 클라이언트와 서버페이지를 개발하였습니다.

프론트엔드 개발

공식 사이트
반응형 UI 개발
  • Media query를 이용한 반응형 UI 구현
heic 확장자 파일 변환 기능 구현
  • 휴대폰으로 영수증 이미지를 업로드 할 때 이미지 확장자가 heic인 아이폰 사용자들을 위해 jpg로 확장자 변환 기능 구현 필요
  • server 페이지에서 라이브러리를 활용하여 jpg 확장자로 변환한 Buffer 파일을 응답 보내주고 클라이언트 페이지에서 base64 형식으로 변환하여 저장
메인 페이지 공지 carousel 구현
  • swiper 라이브러리를 활용하던 중 svelte와의 호환 문제로 충돌 이슈
  • 한 페이지에 두개 이상 사용하면 충돌하는 이슈 발생
  • 라이브러리 대신 Javascript로 setInterval을 활용하여 구현
페이지네이션 구현
  • Javascript로 이전, 다음 버튼을 포함한 페이지네이션 기능 구현
  • api로 총 data count 값을 받아온 후 페이징 UI 구현
관리자 페이지
데이터 순서 변경 버튼 기능 구현
  • find 함수를 사용하여 순서를 변경할 두 개의 index를 찾아 sort값을 변경해주는 로직을 구현하여 사용자의 편의를 높임

백엔드 개발

공식 사이트
서버 api 개발
  • body형식(JSON), cache, header 등을 공통으로 설정할 수 있도록 api 함수를 생성하여 재사용성을 높임
  • DB와 연동하여 데이터 추가, 삭제, 수정 등의 로직을 구현한 후 클라이언트로 응답하는 api 개발
Mysql DB 연동
  • select, insert 등의 query문을 작성하여 mysql DB와 연동
  • 쿼리문에 변수를 활용하여 하나의 요청으로 여러 쿼리문을 실행 가능하게 구현. 특히 검색 기능 구현 시 검색 조건이 담긴 request 데이터들을 변수에 Join하여 저장한 후 쿼리문에 활용하여 검색 기능 구현
페이지네이션 구현
  • 쿼리문을 이용하여 count 값을 전달하고 limit, offset으로 페이징 처리
관리자 페이지
이미지 저장
  • banner 이미지 업로드 시 파일 크기 때문에 base64로 저장하는데 한계가 있어 fs 모듈을 활용하여 서비스 내부 폴더에 저장한 후 url을 추출하여 DB에 저장하는 방식으로 구현
메일 전송
  • nodeMailer 라이브러리를 활용하여 AS 현황 상태(AS 접수, 입고확인, 출고완료 등)가 변경되면 메일을 보내주는 기능 구현
  • 현황 상태는 다중 수정이 가능했기 때문에 id 배열을 받아 id에 따른 데이터를 검색하여 받아온 후 반복문을 사용해 메일 전송 기능 구현
데이터 추가 시 다중 쿼리
  • 순서 변경 기능을 위해 데이터 추가 시 sort 컬럼 값을 이전 데이터들과 비교하여 더 큰 값을 저장해주는 다중 쿼리문을 구현

배운점

Javascript의 다양한 파일 형식
  • 파일 업로드 기능을 구현하면서 Blob, Binary, Buffer 등 새로운 파일 형식들을 알게되었습니다.
Mysql INT 형식 최대값
  • 접수번호 저장시 날짜가 포함된 난수를 만들어 저장하는데 계속 동일한 숫자(2147483647)가 저장되는 것을 확인하였습니다.
  • 숫자형 int에는 최대 허용범위가 존재한다고 합니다. 접수 번호가 max value 보다 큰 수 라서 최대값인 2147483647로 저장되는 것을 알게됐습니다.
    ⇒ 해당 숫자 자릿수보다 작은 수로 저장하여 해결하였습니다.
