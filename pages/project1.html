Catch U Landing Page

랜덤박스 어플 캐치유 랜딩페이지를 의뢰받아 제작하였습니다.
전체 페이지 UI/UX 디자인을 비롯하여 전체 페이지의 클라이언트 구현과 메인의 live 당첨 데이터를 불러오는 서버 구현을 진행하였습니다.

Skill
Next, Context API, SCSS, Tailwind CSS
Role
Front-end, back-end, UI/UX design

프론트엔드 개발

모바일 최적화 UI 개발
  • Tailwind와 SCSS를 활용하여 UI 개발
튜토리얼 구현
  • setTimeout 및 CSS 애니메이션을 활용하여 메인 페이지의 튜토리얼 구현
  • Context API를 활용한 튜토리얼 모드 상태 관리로 페이지 전환
메인 페이지 구현
  • 데이터 불러오는 기능 구현
  • 정품 이미지 더보기 기능 구현
서브 페이지 구현
  • 데이터 불러오는 기능 구현
  • 채용, contact 페이지 폼 전송 기능 구현

백엔드 개발

Live 데이터 불러오는 API 구현
  • Mysql2 라이브러리를 활용하여 API 구현

얻게된 점

배포 서버 환경에서의 Timezone 차이
  • timezone에 따른 시차가 있다는 것은 알고 있었지만 로컬에서 확인 했을 때 별 문제가 없어 그대로 배포를 진행하였습니다. 배포를 하고나니 시간이 잘못 출력되는 것을 확인하여 이유를 파악해보니 서버 환경에 따라 상이할 수 있다는 점을 깨달았습니다. Docker에서 설정을 하면 되는 문제였지만 제대로 이해하기 위해 프론트에서 수정하여 해결하였습니다.
Image 최적화
  • 우회한 도메인의 외부 링크 이미지를 가져올 때 이미지가 제대로 뜨지 않는 현상이 발생하였습니다. Images 컴포넌트를 사용하였는데도 최적화가 되지 않았는데, 외부 링크 이미지는 next config에서 remotePatterns을 미리 설정해주어야 한다는 것을 알게되었습니다.
IOS 환경에 따른 퍼블리싱 차이
  • video 태그 및 애니메이션을 사용하다보니 ios환경에서의 에러를 많이 마주하게 되었습니다. 영상 컨트롤러가 뜨거나 transition이 제대로 적용되지 않는 현상이 발생하였습니다. html의 playsInline 속성과 css will-change 속성 개념을 알게되었습니다.
