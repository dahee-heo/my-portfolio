랜덤박스 어플 캐치유 랜딩페이지를 의뢰받아 제작하였습니다.
전체 페이지 UI/UX 디자인을 비롯하여 전체 페이지의 클라이언트 구현과 메인의 live 당첨 데이터를 불러오는 서버 구현을 진행하였습니다.

프론트엔드 개발

모바일 최적화 UI 개발
  • Tailwind와 SCSS를 활용하여 UI 개발
  • CSS 애니메이션을 활용하여 메인 페이지의 튜토리얼 형식 구현
메인 페이지 구현
  • Api를 통해 데이터 불러오는 기능 구현
  • 정품 이미지 더보기 기능 구현
서브 페이지 구현
  • 채용, contact 페이지 폼 전송 기능 구현
  • API를 통해 데이터 불러오는 기능 구현

백엔드 개발

Live 데이터 불러오는 API 구현
  • Mysql2 라이브러리를 활용하여 API 구현

얻게된 점

배포 환경에서의 Timezone 차이
  • live 데이터를 시간에 따라 몇 분, 몇 시간 전에 당첨된 건지 보여줘야 했는데, 로컬에서 실행시 문제가 없었으나 배포하고 나니 시간이 이상하게 출력 되는 현상이 있었습니다. KST가 UTC와 9시간 차이 난다는 것은 알고 있었지만 로컬과 배포 환경이 다르다는 것을 알게되어 수정하였습니다.
