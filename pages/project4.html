공식 사이트

  • Figma 디자인 소스를 활용하여 전체 페이지 UI 개발
무한 스크롤 개발

얻게된 점

Throttling
  • 요청을 최적화 할 수 있는 debounce와 throttling에 대해 알게 되었고, 두 가지 중 어떤 것을 사용해야할 지 고민하면서 각 각 어떤 경우에 사용하면 효과적인지를 알게 되었습니다.
SQL Query문
  • SQL DB는 처음 접해보았는데 기본 문법과 변수를 활용하여 문장을 작성하는 방법을 처음 알게되었습니다. 이러한 경험을 바탕으로 백엔드 개발자와 협업할 때 더 원활한 커뮤니케이션을 할 수 있을 것 같습니다.
